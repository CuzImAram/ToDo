\doxysubsection{Detailed Description}
A simple To\+Do command-\/line program to manage tasks. 

This program allows the user to\+:
\begin{DoxyItemize}
\item Add, remove, and list tasks.
\item Mark tasks as completed.
\item Reset all tasks.
\end{DoxyItemize}

The tasks are saved in the "{}todo.\+txt"{} file.
