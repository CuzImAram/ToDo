\doxysubsection{Detailed Description}
A simple To\+Do command-\/line program to manage tasks. 

This program allows the user to\+:
\begin{DoxyItemize}
\item Add, remove, and list tasks.
\item Mark tasks as completed.
\item Reset all tasks.
\end{DoxyItemize}

The tasks are saved in the "{}todo.\+txt"{} file.

Loads tasks from a file into the task list. 

This function reads tasks from a file specified by the {\ttfamily FILENAME} constant. Each task in the file is expected to be stored on a new line in the format\+: \begin{DoxyVerb}<completion_status> <task_description>
\end{DoxyVerb}


where\+:
\begin{DoxyItemize}
\item {\ttfamily \texorpdfstring{$<$}{<}completion\+\_\+status\texorpdfstring{$>$}{>}} is a boolean (0 or 1) indicating whether the task is completed.
\item {\ttfamily \texorpdfstring{$<$}{<}task\+\_\+description\texorpdfstring{$>$}{>}} is the string description of the task.
\end{DoxyItemize}

Tasks are loaded into the provided vector, clearing any existing tasks before loading. If the file cannot be opened, a message is displayed to the user.
