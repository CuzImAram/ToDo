 ﻿---\hypertarget{index_autotoc_md0}{}\doxysection{\texorpdfstring{To\+Do Command Line Program}{To\+Do Command Line Program}}\label{index_autotoc_md0}
A simple command-\/line To\+Do application that allows you to manage your tasks. Tasks are stored in a file ({\ttfamily todo.\+txt}), and the program provides commands for adding, removing, marking tasks as done, listing tasks, resetting tasks, and saving them to a file.
